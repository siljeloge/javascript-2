# JavaScript 2 – Social Media App

## Description
To build a social media application using JavaScript and the Noroff API.

## Features
- Register and login users
- View all posts
- View single post
- Create, edit and delete posts
- Follow and unfollow users
- Search posts
- View user profiles
- Logout functionality

## Built with
- JavaScript
- HTML
- CSS
- ES6 Modules
- Noroff Social API
